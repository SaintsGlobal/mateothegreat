## Codebase Patterns
- UI components go in src/components/ui/ and are exported from index.ts
- Use Tailwind CSS with brand colors: brand-coral, brand-cyan, muted, foreground, border
- "use client" directive required for interactive components
- Smooth transitions use duration-300 ease-in-out
- Use &apos; for apostrophes in JSX text (react/no-unescaped-entities rule)
- Pages follow pattern: gradient accent bg, z-10 content, max-w-3xl containers
- For collapsible animations, use CSS Grid with grid-rows-[0fr]/grid-rows-[1fr] instead of JS height measurement (avoids setState in useEffect lint errors)
