# Ralph Progress Log
Started: Fri Mar  6 10:10:39 EST 2026

## Codebase Patterns
- Theme is controlled via "light" class on document.documentElement (html element)
- Use `useSyncExternalStore` for syncing React state with DOM mutations (avoids setState-in-useEffect lint errors)
- Header component is at `src/components/layout/header.tsx`
- UI components go in `src/components/ui/`
- Use blocking script in `<head>` with `dangerouslySetInnerHTML` to prevent flash of wrong theme
- System preference detected via `window.matchMedia('(prefers-color-scheme: light)')`
- Manual theme preference (localStorage) always overrides system preference
